# GenLayer True / False Game

A beginner-friendly custom GenLayer mini-game built in GenLayer Studio.

## Features
- Deployable on GenLayer Studio
- Simple true/false gameplay
- Accepts player answers
- Returns result and score

## Contract File
- `true_false_game.py`

## Example
Statement:
`GenLayer uses AI and blockchain consensus`

Input:
`true`

Output:
- Result = Correct!
- Score = 1

## Purpose
This is a simple mini-game for the GenLayer community.
